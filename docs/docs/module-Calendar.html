convertToRoman(number) → {string}

Converts number 1-3999 (inclusive) to a Roman Numeral for representing the French Republican Calendar year.

Parameters

  • number number
    The year number to convert to roman numeral

Returns

  • string The roman numeral representation of the year passed into the function.

Examples

var romanYear = convertToRoman(5); // romanYear is set to "V"
var romanYear = convertToRoman(25); // romanYear is set to "XXV"
